DisCal - Commands

DisCal uses a simple-to-understand permission scheme for handling access to commands.

Elevated
Requires ADMINISTRATOR or MANAGE_SERVER permission or guild owner
Privileged
Requires discal control role (default control role is @everyone)
Everyone
Everyone will be able to access (unless commands are disabled in the channel)
Patron-Only
Requires guild to be a patron-guild at the early access tier
Dev-Only
Only discal developers are able to use this command

!calendar commands

Alias: !cal !callador

Calendar Commands
Command Description Usage Access
create Starts calendar Wizard !cal create [name] elevated
name Sets calendar's name !cal name [name] privileged
description Sets calendar's description !cal description [desc] privileged
timezone Sets calendar's timezone !cal timezone [tz] privileged
review Displays calendar properties !cal review privileged
confirm Commits the changes made in the wizard !cal confirm privileged
cancel Cancels the wizard !cal cancel privileged
delete Deletes the calendar !cal delete elevated
edit Starts calendar edit wizard !cal edit privileged

Aliases: !e

Event Commands
Command Description Usage Access
create Starts event wizard !event create (name) privileged
summary Sets the event's name !event summary [name] privileged
description Sets the event's description !event description [desc] privileged
start Sets the event's start !event start [yyyy/MM/dd-HH:mm:ss] privileged
end Sets the event's end !event end [yyyy/MM/dd-HH:mm:ss] privileged
color Sets the event's Color !event color (color) privileged
colour Alias of color !event colour (colour) privileged
location Sets the event's location !event location [location] privileged
image Sets the event's image !event image [link] privileged, gif support patron-only
review Displays event properties !event review privileged
confirm Commits changes made in the wizard !event confirm privileged
cancel Cancels the wizard !event cancel privileged
recur Toggles whether the event recurs !event recur [true/false] privileged

count Sets how many times the event recurs. 0 or -1 for infinite. !event count [number] privileged

copy Copies an existing event's details to new event !event copy [event-id] privileged
view Displays an event's details !event view [event-id] everyone
delete Deletes an event !event delete [event-id] privileged

Announcement Commands
Command Description Usage Access
create Starts the announcement create wizard !a create privileged
type Sets the announcement type. Valid types: UNIVERSAL, SPECIFIC, COLOR, RECUR !a type [type] privileged
event Sets the announcement's event. Only needed when using SPECIFIC or RECUR types. !a event [event-id] privileged
color Sets the announcement's color. Only needed when using COLOR type. !a color [color] privileged
colour Sets the announcement's colour. Only needed when using COLOR type. !a colour [colour] privileged
channel Set's the channel it will be posted to !a channel [channel-name] privileged
minutes Sets the minutes before the event to announce. Added to hours !a minutes [number] privileged
hours Sets the hours before the event to announce. Added to minutes !a hours [number] privileged
info Sets the additional info to be posted along with the event !a info [text] privileged
infoOnly Sets if the announcement should only post with info !a infoOnly privileged
public Toggles if the announcement should be cross-posted !a publish privileged, patron-only
review Displays announcement properties !a review privileged
confirm Commits the changes made in the wizard !a confirm privileged
cancel Cancels the announcement wizard !a cancel privileged
edit Starts the announcement editor !a edit [announcement-id] privileged
copy Copies an existing announcement to a new one !a copy [announcement-id] privileged
delete Deletes an announcement !a delete [announcement-id] privileged
enable Enables an announcement !a enable [announcement-id] privileged

view Displays an announcement's properties !a view [announcement-id] everyone
list Lists announcements !a list [number] everyone
sub Subscribes to an announcement to be pinged !a sub [announcement-id] (users/roles) everyone
unsub Unsubscribes to an announcement to be pinged !a unsub [announcement-id] (users/roles) everyone

/settings command

/settings Commands
Command Description Usage Access
view Displays the current settings for the bot in the guild /settings view elevated
role Sets the role required to use privileged commands /settings role [role] elevated
announcement-style Changes what style announcements will be posted as /settings announcement-style [style] elevated
language Changes what language the bot will use in responses /settings language [lang] elevated
time-format Change what format to display date/time when needed /settings time-format [format] elevated
branding Toggles between using discal branding or the guild's name/image where possible /settings branding [true/false] elevated, patron-only
