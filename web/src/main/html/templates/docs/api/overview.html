API Docs - Overview

DisCal provides an extensive and powerful REST API for you to integrate into your website or discord bot.

These docs provide examples of requests, responses, and various info about what is required and returned.


All requests and responses are done via HTTP POST and must be in JSON. All requests must also contain an "Authorization" header with your API key. You can get an API key by contacting the DisCal development team.
